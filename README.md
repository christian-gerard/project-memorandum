# project-memorandum

Memory capturing and sharing for famillies


## Set Up

### Mobile

cd mobile

npm install

npm run web OR npm run ios

### Web

cd web

npm install

npm run dev

### Server

cd server

python -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt

python manage.py runserver

** TURN OFF VIRTUAL ENV WHEN FINISHED

deactivate
